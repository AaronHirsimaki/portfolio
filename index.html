I'm a software developer based in Espoo, Finland, focused building well-designed and meaningful software. I believe in continuous learning, strong fundamentals, and becoming a versatile developer through hands-on experimentation.

My experience spans full-stack web development, mobile apps, and even game development. I use AI-assisted development in my workflow, leveraging tools like ChatGPT, Claude and Claude Code to accelerate development, prototype ideas quickly, and improve problem-solving.

Whether it's a Video Game, React web app, a Spring Boot API, or a mobile experience with React Native, I enjoy the process of turning ideas into polished products. I'm always building, always learning, and aiming to become the kind of developer who can turn any idea into a real, polished product.
